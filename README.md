
# 🍽️ MealMind AI

> Plan Meals. Save Money. Reduce Waste.

MealMind AI is an AI-powered web application that generates personalized daily meal plans, grocery lists, ingredient substitutions, and budget feasibility insights. The platform helps individuals and families make smarter cooking decisions while minimizing food waste and staying within budget.

---

## 🚀 Challenge Overview

### Problem Statement

Build a simple AI micro-app that helps users generate a personal cooking to-do list based on their day.

The application must provide:

- Breakfast Plan
- Lunch Plan
- Dinner Plan
- Grocery List
- Ingredient Substitutions
- Budget Feasibility Logic

---

# 🎯 Solution

MealMind AI transforms daily cooking decisions into a structured and personalized experience.

Users provide:

- Daily Budget
- Family Size
- Dietary Preference
- Available Ingredients
- Cooking Time
- Day Type

The AI then generates:

✅ Breakfast Recommendation

✅ Lunch Recommendation

✅ Dinner Recommendation

✅ Smart Grocery List

✅ Ingredient Alternatives

✅ Budget Analysis
